# Music StoryThis is a repo for managing machine readable artifacts for the [Music Story APIs](http://www.music-story.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Music Story APIs](http://www.music-story.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.music-story.com)- [Developer](http://www.music-story.com)- [Twitter](https://twitter.com/music_story)- [GitHub](https://github.com/music-story)The [APIs.json](https://github.com/api-evangelist/music-story/blob/master/apis.json) index provides links to all the relevant links for the Music Story APIs, including the OpenAPI and Postman if they are present.
